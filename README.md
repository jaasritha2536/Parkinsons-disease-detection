This project uses machine learning to predict whether a person has Parkinson’s disease based on voice features.

Files Included

parkinsons_complete_pipeline.ipynb – Complete ML pipeline

parkinsons_dataset_sample.csv – Sample dataset

parkinsons_full_synthetic.csv – Full synthetic dataset

What the Project Does

Loads and cleans the dataset

Trains different ML models (Logistic Regression, Naive Bayes, SVM, etc.)

Compares model accuracy

Shows results and graphs

Best Model

SVM gave the highest accuracy in this project.

How to Run

Install required Python libraries

Open the Jupyter notebook

Run all cells to see the results
